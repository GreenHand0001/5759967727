# 项目简介

## 核心功能
- 基于 YOLOv8 的电池鼓包损伤检测，包含 7000 张标注数据集（3 类：Battery swollen、Damaged battery、Well battery）
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整源码、训练好的模型及训练结果、运行教程
- 支持自定义模型选择、置信度和 IOU 阈值调整
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果保存功能

## 适用场景
- 电池生产质量控制
- 电池安全检测
- 电池维修前评估
- 电池质量研究
- 工业检测自动化

## 技术特点
- 开发环境：Python 3.9 + Pycharm/VsCode + Anaconda
- 核心技术栈：OpenCV、PyQt5、Ultralytics、PyTorch 1.9
- 基于 YOLOv8 目标检测算法
- 提供图形用户界面，操作简便
- 完整部署包，包含所有依赖环境配置
